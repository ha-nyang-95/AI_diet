# 식품 등의 표시기준 별표 — 알레르기 유발물질 22종

## 표시 의무 22종 항목

식품의약품안전처(MFDS)는 *식품 등의 표시기준* 별표에서 다음 22종을 알레르기 유발물질로 지정하고 가공식품 라벨에 표시 의무를 부여한다: 우유, 메밀, 땅콩, 대두, 밀, 고등어, 게, 새우, 돼지고기, 아황산류, 복숭아, 토마토, 호두, 닭고기, 난류(가금류), 쇠고기, 오징어, 조개류(굴·전복·홍합 포함), 잣, 아몬드, 잔류 우유 단백, 기타. 본 22종은 정의 순서를 보존해야 한다 — 항목 추가·삭제 시 식약처 별표 PDF 갱신과 동기 필수.

## 라벨 표시 규정

가공식품 제조사는 22종 중 1종 이상 함유 시 *원재료* 라벨에 *알레르기 유발물질* 표시를 의무화한다. *대체 처리시설 사용*으로 인한 비의도적 혼입 가능성도 *주의: 알레르기 유발물질이 함유된 제품과 같은 제조시설에서 제조* 등의 문구로 표시한다.

## 일반인 식이 안내

22종 알레르기 유발물질 중 자신에게 해당하는 항목이 있는 사용자는 가공식품 구매 시 라벨 확인을 1차 안전 절차로 한다. 외식·배달 메뉴에서 알레르기 유발물질 함유 여부는 매장에 직접 확인하는 것이 권장된다. 본 BalanceNote 앱은 사용자가 등록한 알레르기 22종을 식단 분석 시 *단락 매칭*으로 자동 경고한다.

## 갱신 SOP — 운영자 분기 확인

식약처 별표 PDF는 비정기적으로 갱신될 수 있어 분기 1회(3월·6월·9월·12월) 운영자가 식약처 공식 사이트에서 최신 별표를 확인하고 본 가이드라인 + ``app/domain/allergens.py`` 22종 lookup을 동기화한다. 동기화 절차는 ``scripts/verify_allergy_22.py`` 분기 SOP로 자동화 검증.

## 인용 안내

본 권고는 *식품 등의 표시기준* (식품의약품안전처) 별표를 요약한다. 알레르기 진단 + 처치는 의료진(알레르기 내과·소아과) 상담 1차 경로이며, 본 가이드라인은 일반인 자가 식이 회피의 참고 자료이다. *(출처: 식품의약품안전처 식품 등의 표시기준)*
